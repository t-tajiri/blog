# Visitor パターンとは

> The purpose of a Visitor pattern is to define a new operation without introducing the modifications to an existing object structure.
(拙訳: 既存のオブジェクトの構造を変更せずに、アルゴリズム・一連の処理を簡単に追加できることを目的とする)
